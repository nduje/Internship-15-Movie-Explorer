# Movie Explorer

### DUMP Internship - React #2

## Task

Build a React app **Movie Explorer** where users can browse movies, search by name, view details, and mark favorites.  
The app uses React hooks, at least one custom hook, and React Router for navigation.

## Features

- **Home** page with title and brief description
- **Movies** page:
    - Search by name
    - Sort and filter results
    - Movie cards with poster, title, year, rating
    - Click card â†’ movie detail page
- **Movie Detail** page:
    - Show title, year, rating, genre, description
    - Add/Remove favorites
    - Back button to previous page
- **Favorites** page:
    - List all favorite movies
    - Remove from favorites
- **404 / NotFound** page for invalid routes

## UI & Behavior

- Loading, error, and empty states
- Hover effects on cards
- Favorites visually marked
- Debounced search
- Scroll to top on route change

## Setup

1. Clone/download repo
2. Install dependencies
3. Run app with Vite + React
4. Navigate pages with React Router links
